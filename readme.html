Project 2

Description

This project is a browser website featuring three images and a search bar. It allows users to perform searches and view the related results.

Installation

To use this project locally, follow these steps:

  1. Clone the repository.
  2. Open the project folder in your preferred code editor.
  3. Start a local server to run the website (e.g., by using Live Server extension in Visual Studio Code).
  4. Access the website through your browser.

Usage

After setting up the project, you can:

  • View the three images displayed on the website.
  • Use the search bar to perform searches and view the related results.
